# **App Name**: BimaSure

## Core Features:

- User Registration: Streamlined registration via email or phone, with a secure password setup.
- Dashboard Overview: A personalized dashboard displaying vehicle profiles, insurance plans, and policy details at a glance.
- Claims Submission: A step-by-step wizard that guides users through submitting claims, including photo uploads and document attachments.
- AI Chat Support: AI-powered chatbot, acting as a tool, trained on insurance FAQs, and providing smart routing to human agents when needed.
- Mobile Money Payment: Instant mobile money payment, with payment history and receipts available.

## Style Guidelines:

- Primary color: Teal (#008080) for trust and reliability.
- Secondary color: Light Gray (#F0F0F0) for clean backgrounds.
- Accent color: Warm Orange (#FFB300) for CTAs.
- Use the Inter font.
- Lucide icons
- 8pt grid for consistent spacing
- Subtle transitions for enhanced engagement
